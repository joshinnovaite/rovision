# ST-06.6: Density-Card Expansion + Snap-To-Zoom

Parent: [[BOW-06-Horizontal-LOD-Playback-Bar]]
Status: done
Progress: done — DensityCard (a cluster marker showing member count) + DensityPopover; clicking a density card runs **snap auto-zoom**: precomputed nearest-neighbour gaps give required `scale = (cardWidth+gap)/gapSeconds`, clamped — zoom until the target becomes a singleton, then snap + select it. Visual confirmation pending.

## Preconditions
- [[ST-06.3-Deterministic-Cluster-Algorithm]] done (clusters + members to expand).
- [[ST-06.4-Recompute-On-Zoom-Imperative-Pan-Play]] done (scale change recomputes clusters).

## Steps
1. Render a cluster as a **DensityCard** (member count / stacked affordance) with a **DensityPopover**
   listing members on hover/expand.
2. **Precompute nearest-neighbour gaps** (seconds) between flags for snap targets.
3. On density-card click, compute **required `scale = (cardWidth + gap) / gapSeconds`** for the target
   flag, **clamp** to the zoom range, and animate/set `scale` so the target **pops out as a singleton**.
4. After the snap, **select** the target track (seek to its `first_frame`, set `selectedTrackId`) so
   overlay + flag highlight in lockstep (reuse the [[ST-05.3-Vertical-Timeline|snap-to-track]] selection).

## Acceptance
- Clicking a density card zooms exactly enough to split it (target becomes a singleton), then snaps
  and selects that track; the popover lists the cluster's members.
