# ST-06.5: Indicator Modes — Tracked / Static

Parent: [[BOW-06-Horizontal-LOD-Playback-Bar]]
Status: done
Progress: done — two indicator modes on the bar: **Tracked** (wheel = scrub; the playhead stays put and time flows under it) and **Static** (wheel = pan the view; playhead rides the timeline). Mode held in the `playbackStore` timeline slice (`mode`); toggled from the toolbar. ctrl+wheel still routes to pinch-zoom ([[ST-06.2-Continuous-Zoom-Gestures]]). Visual confirmation pending.

## Preconditions
- [[ST-06.1-Horizontal-Playback-Bar-Card]] done (bar + `mode` field).
- [[ST-06.2-Continuous-Zoom-Gestures]] done (wheel routing established).

## Steps
1. Add a `mode` field to the timeline slice (`'tracked' | 'static'`) and a **toolbar toggle**.
2. **Tracked** — plain wheel **scrubs** (`deltaSeconds = deltaX / pxPerSecond` → set `<video>`
   `currentTime`); the playhead is anchored and the timeline flows under it.
3. **Static** — plain wheel **pans** the view (`viewStartTime`); the playhead rides the timeline at
   its true time position.
4. Keep **ctrl+wheel → pinch-zoom** in both modes; mode only switches the **plain-wheel** meaning.

## Files
- `app/frontend/src/components/playback/PlaybackBarCard.tsx` (wheel handler branches on `mode`),
  `app/frontend/src/components/playback/PlaybackToolbar.tsx` (mode toggle),
  `app/frontend/src/state/playbackStore.ts` (`mode`).

## Acceptance
- In Tracked the wheel scrubs against a fixed playhead; in Static the wheel pans the view; ctrl+wheel
  zooms in both; toggling the mode swaps the behaviour live.
