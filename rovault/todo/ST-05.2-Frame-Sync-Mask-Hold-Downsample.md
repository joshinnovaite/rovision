# ST-05.2: Frame-Sync, Mask-Hold, Down-sample

Parent: [[BOW-05-Playback]]
Status: done
Progress: done — useFrameSync rAF loop maps currentTime->frame (round(t*enc_fps)); mask-hold via heldPolygonsAt binary search; detector/SAM2 sliders down-sample by frame-quantised box/mask lookup. NOTE: pragmatic React re-render per frame (~30/s, few shapes) instead of imperative ref mutation; imperative path is the documented fallback if jank appears.

## Preconditions
- [[ST-05.1-Video-Stage-And-SVG-Overlay]] done (video + SVG overlays render).
- [[ST-03.3-Stores-And-Derived-Structures]] done (`FrameIndex`, `HeldMaskTimeline`, down-sample util).

## Steps
1. Build `useFrameSync`: a `requestAnimationFrame` loop reading the `<video>` `currentTime`.
2. Compute `currentFrame = round(currentTime · enc_fps)` (use **`enc_fps`**, not `fps`, to avoid drift)
   and write **one** store field (`playbackStore.currentFrame`).
3. `OverlayLayer` subscribes **imperatively** via `useStore.subscribe` and mutates shape refs
   directly — **zero React renders in the ~50fps hot path** (the key perf risk).
4. On each frame, resolve current detection boxes from `FrameIndex` and update box refs.
5. Resolve the **held** polygon per track via binary search over `HeldMaskTimeline` change-points
   (DR-009); clear a track's mask after its `last_frame`.
6. Wire live down-sample: apply the quantization util (detector/refine everyN) so masks held longer
   when refine frames are dropped; **clamp to native cadence**; no refetch.
7. Verify with the React profiler that no component re-renders per frame during playback.

## Files
- `app/frontend/src/hooks/useFrameSync.ts`; updates to
  `app/frontend/src/components/OverlayLayer.tsx` (imperative subscribe + ref mutation).

## Acceptance
- Boxes + masks track playback and **do not blink** between refine frames (mask-hold works).
- Adjusting frequency sliders changes shown cadence live without refetch (clamped to native).
- React profiler shows **no per-frame re-render** at ~50fps.
