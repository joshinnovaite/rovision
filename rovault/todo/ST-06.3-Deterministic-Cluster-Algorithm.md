# ST-06.3: Deterministic Cluster Algorithm

Parent: [[BOW-06-Horizontal-LOD-Playback-Bar]]
Status: done
Progress: done — `lib/timelineClusters.ts`: deterministic **two-pass** clusterer — (1) greedy proximity grouping in **pixel space** (grid-independent), (2) **mean-centred de-overlap relaxation** enforcing `pos[i] ≥ pos[i-1] + cardWidth + gap`. Provably non-overlapping, no hysteresis. Replaces the original fixed-bin + mean-shift idea. Unit-tested in `lib/timelineClusters.test.ts` (non-overlap, determinism, monotonic singletons). Visual confirmation pending.

## Preconditions
- [[ST-06.1-Horizontal-Playback-Bar-Card]] done (flags positioned by `first_frame` × `scale`).

## Steps
1. Implement a **pure, in-repo** clusterer (~30 lines; **no clustering dependency**) taking flag
   pixel-positions + `cardWidth`/`gap` and returning clusters with final positions.
2. **Pass 1 — greedy proximity grouping** in **pixel space** (so it is grid-independent; markers do
   not jump as they cross a bin edge).
3. **Pass 2 — mean-centred de-overlap relaxation**: centre each cluster on its members' mean, then a
   single monotone sweep enforcing `pos[i] ≥ pos[i-1] + cardWidth + gap`. Provably non-overlapping;
   singleton pop-out emerges automatically as `scale` rises.
4. **Recompute from scratch each zoom level** (no carried state → no hysteresis/flicker).
5. Unit-test: **non-overlap**, **determinism** (same scale → same output), **monotonic singletons**.
