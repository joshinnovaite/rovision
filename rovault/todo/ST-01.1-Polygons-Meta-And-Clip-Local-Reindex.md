## Preconditions
- §17 of `rovision/notebooks/subsea_defect_demo.ipynb` already exports `detection_cards.json`
  (per-frame cards with `area_px`/`coverage_frac` on refine frames) and `track_cards.json`.
- The two-pass structure exists: Pass 1 buffers frames to `TMP/%05d.jpg`, Pass 2 overlays.
- Edit the cell programmatically via a Python script that loads the notebook JSON (do NOT hand-edit
  the `.ipynb`); the harness runs on Colab GPU, but the cell *source* edit is done locally.

## Steps
1. Add `mask_to_polygons(mask_bool, W, H)` using `cv2.findContours` + `cv2.approxPolyDP`.
2. Set `epsilon = 0.004 * cv2.arcLength(contour, True)`; filter contours with area `< 0.0008 * (W*H)`.
3. Keep the largest `<= 4` contours; normalize all points to `0..1`, round to 4 dp; return list-of-rings.
4. On each **refine frame** card, attach `card['polygons']` alongside the existing `area_px`/`coverage_frac`.
5. Re-index cards to **clip-local 0-based**: card `frame` = `rec['k']` (clip frame), no source offset.
6. Confirm `track_cards` first/last frames follow from the re-indexed cards automatically.
7. Build `meta.json` dict with fields: `schema_version, source_video, asset, fps,
   enc_fps=round(fps), start_frame, n_frames, width, height, classes[15], refine_every,
   min_track_len, frame_indexing:'clip_local_0based'`.
8. Assert the invariant `n_frames == len(per_frame)` (raise if violated — guards dedup/seeking later).
9. Write `meta.json` into the bundle directory next to the JSON cards.
