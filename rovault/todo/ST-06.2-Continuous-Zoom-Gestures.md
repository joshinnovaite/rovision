# ST-06.2: Continuous Zoom Gestures

Parent: [[BOW-06-Horizontal-LOD-Playback-Bar]]
Status: done
Progress: done — continuous zoom via @use-gesture/react pinch (about cursor) + zoom buttons in the toolbar; ctrl+wheel routed to pinch; wheel otherwise scrubs (Tracked) / pans (Static) per [[ST-06.5-Indicator-Modes-Tracked-Static]]. Zoom updates `scale` only (clusters recompute on scale change, [[ST-06.4-Recompute-On-Zoom-Imperative-Pan-Play]]). Visual confirmation pending.

## Preconditions
- [[ST-06.1-Horizontal-Playback-Bar-Card]] done (bar + timeline slice exist).

## Steps
1. Add **@use-gesture/react** to the bar: **pinch zooms about the cursor** (anchor the time under the
   cursor so it stays put).
2. Add toolbar **zoom in/out buttons** that step `scale` (continuous, clamped to a min/max range).
3. Route **ctrl+wheel → pinch**; plain wheel is handled by the mode in
   [[ST-06.5-Indicator-Modes-Tracked-Static]] (scrub vs pan), not zoom.
4. Write **only `scale`** on zoom — pan offset (`viewStartTime`) is a separate concern; cluster
   recompute keys off `scale` ([[ST-06.4-Recompute-On-Zoom-Imperative-Pan-Play]]).

## Files
- `app/frontend/src/components/playback/PlaybackBarCard.tsx` (gesture binding),
  `app/frontend/src/components/playback/PlaybackToolbar.tsx` (zoom buttons),
  `app/frontend/src/state/playbackStore.ts` (`scale`).

## Acceptance
- Pinch and the zoom buttons smoothly change zoom; the time under the cursor stays anchored on pinch;
  ctrl+wheel zooms while plain wheel does not.
