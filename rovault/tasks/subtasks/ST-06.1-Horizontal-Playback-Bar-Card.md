# ST-06.1: Horizontal Playback Bar Card

Parent: [[BOW-06-Horizontal-LOD-Playback-Bar]]
Status: done
Progress: done — PlaybackBarCard mounted beneath the video stage; landscape-oriented horizontal bar replacing the removed VerticalTimeline. Reads `tracks` (`first_frame`) and positions one flag per non-omitted track along a time axis; artefacts dimmed, asset lens dims off-asset further. VideoHeader + PlaybackToolbar frame it. Visual confirmation pending.

## Preconditions
- [[ST-05.1-Video-Stage-And-SVG-Overlay]] done (video stage exists to sit the bar beneath).
- [[ST-03.3-Stores-And-Derived-Structures]] done (`track_cards` / timeline geometry available).

## Steps
1. Add a **horizontal** playback-bar-card component below the video and **remove** `VerticalTimeline.tsx`.
2. Introduce a `playbackStore` **timeline slice** (`scale`, `mode`, `viewStartTime`, `barWidthPx`,
   `durationSec`, `encFps`, `playbackRate`); **retire `windowSeconds`**.
3. Position one flag per **non-omitted track** by `first_frame` time across `barWidthPx`; the bar reads
   **`tracks`, not detections**, so down-sample sliders never move a flag.
4. Dim **artefacts** by default; let the **asset lens** dim off-asset tracks further (reuse the
   [[ST-05.4-Asset-Behaviours-And-Playback-Work-Orders|asset]] dimming).
5. Frame the bar with `VideoHeader` and `PlaybackToolbar`.
