# ST-05.3: Vertical Timeline

Parent: [[BOW-05-Playback]]
Status: done
Progress: done — VerticalTimeline: playhead at midpoint, FlagCards positioned by first_frame time translating with playback, ResizeObserver-measured px/sec, non-passive wheel = scrub, click card = snapToTrack (select + seek). Visual confirmation pending.

## Preconditions
- [[ST-05.2-Frame-Sync-Mask-Hold-Downsample]] done (`currentFrame`/`currentTime` synced in store).
- [[ST-03.3-Stores-And-Derived-Structures]] done (timeline geometry util; `track_cards`).

## Steps
1. Build the vertical timeline: `pxPerSecond = trackHeight / windowSeconds`; playhead **fixed at
   midpoint**.
2. Render one **card per tracked instance** (`track_id`); card vertical offset =
   `midpointY + (cardTime − currentTime)·pxPerSecond` (cards translate as playback moves).
3. `wheel` / drag → scrub: `deltaSeconds = deltaY / pxPerSecond` → set the `<video>` `currentTime`.
4. Click card → `snapToTrack`: set `selectedTrackId` + seek to that track's `first_frame`.
5. Highlight the selected card **and** overlay shapes from the one `selectedTrackId` (lockstep).
6. Filter/dim cards by `settingsStore` (omittedClasses, asset) — artefacts dimmed by default.

## Files
- `app/frontend/src/components/Timeline.tsx`, `app/frontend/src/components/FlagCard.tsx`,
  helper `snapToTrack` in `playbackStore` or a util.

## Acceptance
- Cards scroll smoothly past the fixed midpoint playhead as playback advances.
- Scrolling/dragging scrubs the video; clicking a card snaps to its `first_frame` and selects it;
  card + overlay highlight move in lockstep.
