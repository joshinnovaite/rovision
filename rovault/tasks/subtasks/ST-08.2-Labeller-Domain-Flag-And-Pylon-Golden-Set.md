## Preconditions
- Pylon class list locked into the registry ([[ST-08.1-Pylon-Footage-And-Class-Proposal]]).
- Labeller currently hardcodes the subsea taxonomy ([[label_defects.py]]).

## Steps
1. Add a `--domain` flag to `label_defects.py`; load classes from `rovision/domains.json` for the chosen domain (retire the hardcoded subsea copy). **[done]** — the class list, default footage dir, and default output path are all registry-derived.
2. Route output to a domain-specific golden set: `labels_pylon.json` (keeping `labels.json` for subsea). **[done]** — pylon resolves to `test_footage_pylon/labels_pylon.json`; subsea unchanged.
3. Run multi-pass labelling over the pylon footage to build the pylon golden seed set. **[done]** — `labels_pylon.json`, 193 boxes / 4 clips / all 5 classes; no degenerate boxes.

## Files
