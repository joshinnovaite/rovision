# ST-03.3: Stores And Derived Structures

Parent: [[BOW-03-Frontend-Foundation]]
Status: done
Progress: done — Zustand stores (video/playback/settings) + frameIndex.ts (FrameIndex + HeldMaskIndex with binary-search mask-hold) + downsample.ts. Verified end-to-end: SPA boots, library loads via proxy, clip range returns 206 through Vite.

## Preconditions
- [[ST-03.2-Types-API-Client-Fixture]] done (types + api client + fixture available).

## Steps
1. Create `videoStore` (Zustand): `hash`, `meta`, `tracks`, prebuilt `FrameIndex`,
   `HeldMaskTimeline`, `library`.
2. Create `playbackStore` (hot path): `currentFrame`, `isPlaying`, `selectedTrackId`,
   `windowSeconds`. The `<video>` owns `currentTime`; `currentFrame` is a derived mirror.
3. Create `settingsStore`: `asset`, `omittedClasses`, `detectorEveryN`, `refineEveryN`.
4. Build `FrameIndex` — per-frame lookup of detection cards keyed by clip-local frame.
5. Build `HeldMaskTimeline` — per-track refine-frame **change-points** (sorted) for mask-hold
   resolution by binary search; cleared after each track's `last_frame` (DR-009).
6. Add geometry utils: box scaling against source-pixel viewBox; polygon stays normalized `0..1`.
7. Add a `timeline geometry` util: `pxPerSecond = trackHeight / windowSeconds`, card offset =
   `midpointY + (cardTime − currentTime)·pxPerSecond`.
8. Add a down-sample util: keep detection iff `frame % detectorEveryN === 0`; keep refine iff its
   refine-index is a kept multiple (else mask-hold persists longer); **clamp to native cadence**.
9. Add unit tests for `HeldMaskTimeline` binary search, geometry, and down-sample quantization.

## Files
- `app/frontend/src/stores/{videoStore,playbackStore,settingsStore}.ts`.
- `app/frontend/src/lib/{frameIndex,heldMask,geometry,downsample}.ts` (+ `*.test.ts`).

## Acceptance
- Stores initialise from fixture; `FrameIndex`/`HeldMaskTimeline` build from fixture cards.
- Binary-search mask resolution returns the correct held polygon between refine frames; clears after
  `last_frame`. Down-sample never *increases* shown cards and clamps to native cadence. Tests pass.
